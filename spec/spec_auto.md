# Requirement ID: FR_auto_1
- Description: [The system shall allow users to track their emotions and moods over time.]
- Source Persona: [Engaged and Hopeful User]
- Traceability: [Derived from review group G1]
- Acceptance Criteria: [Given a user has logged in, When they navigate to the tracking section, Then they can select and record their current emotions and moods.]
